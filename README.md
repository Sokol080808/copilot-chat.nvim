# copilot-chat.nvim 💬

A Neovim plugin that brings a VS Code-like Copilot Chat interface to your favorite editor.

> **Note:** The code for this initial plugin structure was generated by a neural network.

*(Note: This project connects directly to GitHub Copilot's `gpt-4o` chat endpoints!)*

## ✨ Features

* **Real GitHub Copilot Integration**: Automatically detects your official Copilot token or prompts you through the OAuth Device Flow if missing.
* **VS Code-like UI**: A familiar vertical split for chat history and an integrated prompt input buffer.
* **Live Token Streaming**: Real-time token-by-token streaming of the AI's response without blocking your editor.
* **Markdown Support**: Chat history renders with Neovim's native Markdown highlighting.
* **Native Lua**: Written entirely in Lua using Neovim's modern API and `curl`.

## 📋 Prerequisites

* Neovim >= **0.8.0**

## 🚀 Installation

Using [lazy.nvim](https://github.com/folke/lazy.nvim):

```lua
{
    "Sokol080808/copilot-chat.nvim",
    dependencies = {
        -- Future dependencies
    },
    opts = {
        -- Optional configuration goes here
    },
    config = function(_, opts)
        require("copilot-chat").setup(opts)
    end,
}
```

Using [packer.nvim](https://github.com/wbthomason/packer.nvim):

```lua
use({
    "Sokol080808/copilot-chat.nvim",
    config = function()
        require("copilot-chat").setup()
    end,
})
```

## 🎮 Usage

1. Open the chat window with the command:
   ```vim
   :CopilotChat
   ```
2. Type your prompt in the bottom input buffer.
3. Submit your prompt:
   - Press `<Enter>` in **Normal mode**
   - Press `<C-s>` in **Insert mode**

## 🛠️ Configuration

You can configure the plugin by passing a table to the `setup` function. The table is currently empty but will support API configuration and window sizing in the future.

```lua
require("copilot-chat").setup({
    -- Future options:
    -- window_width = 50,
    -- model = "gpt-4o",

```
