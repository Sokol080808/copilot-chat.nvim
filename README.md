# copilot-chat.nvim 💬

A Neovim plugin that brings a VS Code-like Copilot Chat interface to your favorite editor.

> **Note:** The code for this initial plugin structure was generated by a neural network.

*(Note: This project uses the official GitHub Models API endpoint.)*

## ✨ Features

* **Official GitHub Models API**: Uses the documented `https://models.github.ai/inference/chat/completions` endpoint.
* **VS Code-like UI**: A familiar vertical split for chat history and an integrated prompt input buffer.
* **Live Token Streaming**: Real-time token-by-token streaming of the AI's response without blocking your editor.
* **Markdown Support**: Chat history renders with Neovim's native Markdown highlighting.
* **Native Lua**: Written entirely in Lua using Neovim's modern API and `curl`.

## 📋 Prerequisites

* Neovim >= **0.8.0**
* GitHub token with `models` scope (PAT)

Create one at: https://github.com/settings/tokens

Then set one of these environment variables before opening Neovim:

```bash
export GITHUB_TOKEN="your_token_here"
# or
export GH_TOKEN="your_token_here"
```

## 🎮 Usage

1. Open the chat window with the command:
   ```vim
   :CopilotChat
   ```
2. Type your prompt in the bottom input buffer.
3. Submit your prompt:
   - Press `<Enter>` in **Normal mode**
   - Press `<C-s>` in **Insert mode**

### 🔐 Login Flow (Official)

This plugin uses official GitHub Models authentication.

Token discovery order:
1. `GITHUB_TOKEN`
2. `GH_TOKEN`
3. `gh auth token` (from GitHub CLI login)

If no token is found, the plugin automatically opens a terminal split and starts:

```bash
gh auth login -h github.com -s models:read -w
```

You can also run this manually from Neovim:

```vim
:CopilotChatLogin
```
